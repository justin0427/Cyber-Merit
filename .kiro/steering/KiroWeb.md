## 1. 技術堆疊 (Tech Stack)
- **Framework:** React (Functional Components + Hooks).
- **Styling:** Tailwind CSS (使用 utility classes).
- **Icon:** 使用 `lucide-react` 或 FontAwesome (CDN).
- **Animation:** CSS Animations 或 Framer Motion.

## 2. 風格與限制 (Constraints)
- **RWD:** 必須是響應式設計 (Mobile First)，確保在手機上也能完美操作。
- **單一檔案:** 為了快速原型開發，請將 CSS/JS/HTML 全部整合在一個 `index.html` 中 (使用 Babel standalone)。
- **配色:** Cyberpunk 風格 (Neon Green, Dark Background, Glitch effects).
- **語言:** 註解與對話使用繁體中文。
